===============DEPENDS function=================   
in a way, Depends() acts like middleware
How Depends() works :
    Depends() is a FastAPI dependency injection system.
    It runs the function you pass to it (like get_current_user or get_optional_user) before your route function executes.
    Whatever that dependency function returns is injected as a parameter into your route.

Why it feels like middleware : 
    Like middleware, it runs before your route logic.
    It can perform checks, e.g., authenticate the user, validate input, or raise errors.
